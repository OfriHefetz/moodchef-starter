# Git & GitHub Self Practice

Welcome! In this exercise, you will practice the basic Git and GitHub workflow.

You will:
1. Fork this repository
2. Clone your fork to your computer
3. Make small changes to the project
4. Save your changes using Git commits
5. Push your work back to GitHub

## What are Git and GitHub?

Git is a version control tool. It helps you save the history of your project.

GitHub is a website where Git repositories can be stored online and shared with others.

A commit is like a save point in a game.
